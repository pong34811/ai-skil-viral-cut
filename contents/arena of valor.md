# Arena of Valor (RoV)

> MOBA มือถือยอดนิยม พัฒนาโดย TiMi Studio Group, เผยแพร่โดย Garena ในไทย

## ภาพรวม
- **แนวเกม:** Multiplayer Online Battle Arena (MOBA) 5v5
- **ผู้พัฒนา:** TiMi Studio Group (Tencent)
- **ผู้ให้บริการไทย:** Garena
- **แพลตฟอร์ม:** Android, iOS, iPadOS
- **ความยาวต่อแมทช์:** ~10-15 นาที
- **ภาษา:** อังกฤษ, ไทย และอื่น ๆ

## จุดเด่นของเกม
- เลน: Top (DS), Jungle, Mid, Bot (Abyssal), Support (Roam)
- ฮีโร่แบ่งตามบทบาท: Tank, Warrior, Assassin, Mage, Marksman, Support
- ระบบ Rank, แผนที่ Classic, แผนที่ Rank
- อีเวนต์พิเศษ อีสปอร์ต (RoV Pro League)

## แผนที่ / โหมดยอดนิยม
- **5v5 Rank / Classic** — แผนที่มาตรฐาน 3 เลน
- **3v3** — แผนที่ย่อ
- **Arcade** — โหมดสนุก (Clone War, One Shot, ฯลฯ)
- **Carano Chess** — โหมด Auto Chess

## จุดที่มักเกิด Viral Moment
- **Dive / Tower Dive** — พุ่งเข้าฆ่าในป้อม
- **Steal Boss** — ขโมย Dark Slayer / Abyssal Dragon
- **Teamfight แมส** — 5v5 แตกลุ้น
- **Savage / Multi-kill** — ฆ่า 4-5 คนรวด
- **Feed / Troll** — เพื่อนร่วมทีมเล่นแย่ หรือเล่นตลก
- **Flip / Comeback** — พลิกเกมจากแพ้เป็นชนะ
- **แซวเพื่อน / พูดคุย** — โต้ตอบสนุกระหว่างเพื่อนร่วมทีมหรือผู้ชม
- **ปิงแมพ / วางแผน** — ช่วงวางแผนก่อนทีมไฟต์
