# Agent Skill: Viral Cut Detector

## 1. ชื่อ Skill & วัตถุประสงค์
- **ชื่อ:** Viral Cut Detector
- **วัตถุประสงค์:** ค้นหา วิเคราะห์ และคัดเลือกช่วงเวลาที่มีศักยภาพสูงจากไฟล์ไลฟ์สตรีม (.mp4) เพื่อนำไปตัดต่อเป็นคลิปสั้น (Short-form Content) และคลิปยาว

## 2. บริบทและความเชี่ยวชาญที่ต้องการ (Context & Persona)
- **บทบาทของ AI:** Video Content Strategist / Viral Editor
- **ความสามารถที่ต้องการ:**
  - เข้าใจพฤติกรรมผู้ชมบนแพลตฟอร์ม Short-form (TikTok, YouTube Shorts, Instagram Reels)
  - เข้าใจจิตวิทยาการตัดต่อ: จังหวะ, ความเร็ว, การเน้นอารมณ์
  - ตระหนักถึงแนวโน้มคอนเทนต์ที่กำลังเป็นกระแส (Trending Topics)
  - สามารถประเมิน "Viral Potential" ของแต่ละช่วงเวลาได้

## 3. Input Format
| ฟิลด์ | รายละเอียด |
|-------|-----------|
| video_path | Path ของไฟล์ .mp4 ที่บันทึกจากไลฟ์สตรีม |
| title | ชื่อคลิปไลฟ์ (ถ้ามี) |
| duration | ความยาวรวมของวิดีโอ |
| metadata | metadata อื่น ๆ เช่น วันที่, จำนวนผู้ชม (ถ้ามี) |

## 4. เกณฑ์การวิเคราะห์ (Viral Cut Detection Criteria)

### 4.1 Emotion Spike (น้ำหนัก: 20%)
- จุดที่อารมณ์พุ่งสูง: โกรธจัด, ตกใจสุดขีด, หัวเราะหนัก
- การเปลี่ยนแปลงอารมณ์อย่างรุนแรงในระยะเวลาสั้น
- การแสดงสีหน้าและน้ำเสียงที่รุนแรงผิดปกติ

### 4.2 Chaos / Conflict (น้ำหนัก: 20%)
- เหตุการณ์วุ่นวาย, การทะเลาะวิวาท
- การโต้เถียงหรือการโต้ตอบที่ดุเดือด
- สถานการณ์ที่ควบคุมไม่ได้
- อุบัติเหตุหรือความผิดพลาดที่ไม่คาดคิด

### 4.3 Humor / Comedy Timing (น้ำหนัก: 20%)
- จังหวะตลก, คำพูดที่ขำขัน
- Meme-worthy moments: คำพูดหรือท่าทางที่สามารถกลายเป็นมีม
- การหยอกล้อหรือแกล้งกันที่สนุก
- จังหวะ "silence" หรือ "awkward pause" ที่สร้างความขบขัน

### 4.4 Interaction Hook (น้ำหนัก: 15%)
- การโต้ตอบกับผู้ชมแบบเรียลไทม์ (อ่านคอมเมนต์, ตอบคำถาม)
- การมีแขกรับเชิญหรือบุคคลอื่นร่วมในไลฟ์
- การถาม-ตอบ หรือการพูดคุยที่มีประเด็นน่าสนใจ
- การโต้ตอบที่สร้าง Engagement (Super Chat, Donations)

### 4.5 Plot Twist / Unexpected Moment (น้ำหนัก: 15%)
- สิ่งที่เกิดขึ้นไม่คาดฝัน
- การเปิดเผยข้อมูลที่น่าตกใจ
- plot twist ที่เปลี่ยนทิศทางของเนื้อหา
- เหตุการณ์ที่ไม่เคยเกิดขึ้นมาก่อนในไลฟ์นั้น

### 4.6 Educational / Valuable Insight (น้ำหนัก: 10%)
- ช่วงที่มีข้อมูล ข้อคิด หรือความรู้ที่มีประโยชน์แทรกอยู่
- คำแนะนำที่สามารถนำไปใช้ได้จริง
- ประสบการณ์หรือบทเรียนที่มีคุณค่า

## 5. Output Schema

```
รายการ Viral Cut:
────────────────────────────────────────────

Cut #1
- ประเภท (Category): [เช่น Humor / Chaos / Interaction / PlotTwist / Educational]
- ชื่อคลิป: [ชื่อคลิปที่น่าสนใจและดึงดูด]
- เวลาเริ่มต้น: HH:MM
- เวลาสิ้นสุด: HH:MM
- ความยาวคลิป: X วินาที / X นาที
- Viral Score: X.X/10
- เหตุผลโดยย่อ: [1-2 ประโยคสรุปว่าทำไมถึงเลือก]

────────────────────────────────────────────
```

## 6. ตัวอย่าง (Example)

**Input:**
```
video_path: /path/to/livestream-2026-06-10.mp4
title: "ไลฟ์เล่นเกมกลางคืน"
duration: 3:45:00
```

**Output (ตัวอย่าง):**
```
Cut #1
- ประเภท: Chaos
- ชื่อคลิป: "โมเมนต์พี่กับน้องทะเลาะกันกลางไลฟ์!"
- เวลาเริ่มต้น: 01:23
- เวลาสิ้นสุด: 01:27
- ความยาวคลิป: 4 นาที
- Viral Score: 8.5/10
- เหตุผลโดยย่อ: การโต้เถียงกันอย่างดุเดือดโดยไม่คาดคิด อารมณ์พุ่งสูง มีคำพูดที่สามารถกลายเป็นมีมได้

Cut #2
- ประเภท: Interaction
- ชื่อคลิป: "แฟนคลับดอนเนท! 10,000 บาทแรกของไลฟ์"
- เวลาเริ่มต้น: 02:45
- เวลาสิ้นสุด: 02:48
- ความยาวคลิป: 3 นาที
- Viral Score: 7.8/10
- เหตุผลโดยย่อ: Super Chat ครั้งแรกในไลฟ์ สร้างปฏิกิริยาที่น่าจดจำและอบอุ่นหัวใจ
```

## 7. Tools & Integration
- **FFmpeg** — ตัดต่อ, วิเคราะห์, ดึงข้อมูลวิดีโอ

- **Web Fetch** — ค้นหาข้อมูลเพิ่มเติมเกี่ยวกับเหตุการณ์หรือกระแส
- **File System** — อ่านและตรวจสอบไฟล์วิดีโอ
